Basic Tags

Tags Description
<!DOCTYPE html> Defines the Document type
<html> Defines html Document / root element of HTML Document
<head> Contain Metadata [document title, character set, styles, links, scripts]
<title> Title for The Document
<body> Defines Visual parts of the document
<p> For Creating Paragraph
<h1> to <h6> To create headings
<hr> Defines Horizontal line / themantic change
<br> Single line break
<!--,,,--> Comments in HTML

Text Formatting

Tags Description
<i> Make text Italic
<em> Emphasized text / italic
<b> Make text Bold
<strong> Important text / bold
<u> Make text Underlined
<del> Deleted text
<ins> Inserted text
<mark> Mark as Highlighted text
<pre> Preformatted text / display as written in Text Editor
<q> Add quotation mark
<small> Smaller text
<big> Bigger Text
<sub> add text as subscript
<sup> add text as supercript
<abbr> abbreviation or an acronym
<cite> Add Title of a work
<code> used to add piece of computer code
<meter> Scalar measurement within a known range
<progress> Defines progress


Form , Inputs

Tags Description
<form> Defines form to get user inputs
<input> Add an input control
<button> Clickable button
<select> Create Drop-down list
<option> Add options in Drop-down list
<optgroup> Defines your options group in Drop-down list
<textarea> Multiline input control
<datalist> show specific list options for input controls
<label> add labels for input control
<fieldset> Defines group related element in your form
<legend> write caption for fieldset element


Images

Tags Description
<img> add images in your web page


Audio & Video

Tags Description
<audio> to add music/sound in web page
<video> to add video in web page
<source> used to add multiple media resources for [audio , video , picture]


List

Tags Description
<ul> use to create unordered list
<ol> use to create ordered list
<li> Add list items in ordered list and unordered list
<dl> add description type lists
<dt> defines terms in your description list
<dd> add description of description terms / name


Table

Tags Description
<table> create table for table data
<tr> to create rows in table
<td> defines a table cell
<thead> to group header content
<tbody> to group body content
<tfoot> to group footer content
<caption> create caption for table


Link

Tags Description
<a> create hyper link
<link> to link external resources [mostly use to link stylsheet]
<nav> to add navigation links


Frames

Tags Description
<iframe> create inline frame in your webpages


Styles & Semantics element

Tags Description
<header> defines header
<footer> defines footer
<section> add section in web page
<article> to add article
<details> use to add additional details that the user can show or hide
<summary> write visible heading for a <details> element
<aside> content aside from the web page content
<style> content aside from the web page content
<div> to create block level section in document
<span> to create inline section in document
<main> specifies the main content
